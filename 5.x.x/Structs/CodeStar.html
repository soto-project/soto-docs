    Parameters

    AWSClient used to process requests
    region

    Region of server you want to communicate with. This will override the partition parameter.
    partition

    AWS partition where service resides, standard (.aws), china (.awscn), government (.awsusgov).
    endpoint

    Custom endpoint URL to use instead of standard AWS servers
    timeout

    Timeout value for HTTP requests
